Projects
The Sound of Personality (UNIHACK 2023)
This is my first hackathon, and also my first time using APIs. Our team was a bit disorganized at first as we had no prior experience with hackathons, but we managed make a web app to analyze a person's Spotify listening history and estimate their Myers-Briggs personality type using data from 16personalities and Spotify.

Negative Base Converter (ALPHA)
This is a web app that converts numbers from any base to any other base, including negative bases, which have the unique property of being able to represent all positive and negative integers without using a negative sign. As this topic is not very popular, I had to do a lot of research in order to discover an algorithm that can deal with the conversion properly especially when it comes to fractional numbers.

allRGB
This is part of a bigger project started by "deidee", which has the objective to create images with one pixel for every RGB color, without any duplicate or missing colors. I aimed to create a Python program that would take any image and convert it into an allRGB image. As this was my first programming project ever, my program used the brute-force solution, and although functional, it was quite inefficient. Now that I am more familiar with algorithm design and data structures, I plan to rewrite the program to make it more efficient.
